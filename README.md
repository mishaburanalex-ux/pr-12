## 🎯 Мета роботи
Скласти програму для обчислення найбільшого спільного дільника (НСД) двох чисел за алгоритмом Евкліда та найменшого спільного кратного (НСК).

## 📖 Теоретичні відомості
**Алгоритм Евкліда (через ділення):**
Поки B ≠ 0, встановити T = A % B, потім A = B, B = T. Коли B = 0, НСД = A (з урахуванням модулю).

**Формула для НСК:**
НСК(A, B) = |A × B| / НСД(A, B)  (якщо один з аргументів 0, то НСК = 0)

## 📝 Приклади розв'язання задач
### Задача 1
**Умова:** Маємо 60 цукерок і 45 яблук. Яку найбільшу кількість однакових подарунків можна зібрати?

**Розв'язок:** НСД(60, 45) = 15 → 15 подарунків (по 4 цукерки і 3 яблука в кожному).

### Задача 2
**Умова:** Два теплоходи повертаються через 10 і 18 діб. Через скільки діб вони зустрінуться?

**Розв'язок:** НСК(10, 18) = 90 → через 90 діб.

## 🔧 Алгоритм роботи покращеної програми (pr12_2.py)
1. Отримати два цілі числа (через argv або input()).
2. Обробити від'ємні значення (взяти модуль) та перевірити випадки з нулем.
3. Використати алгоритм Евкліда через операцію `%` для швидкого знаходження НСД.
4. Обчислити НСК через формулу: НСК = |A × B| / НСД (якщо НСД ≠ 0).
5. Вивести результати.
